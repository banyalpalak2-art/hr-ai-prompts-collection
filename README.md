 HR AI Class Demonstration

 Overview

This repository showcases the practical use of **Artificial Intelligence in Human Resources (HR)** through creative prompts and visual outputs.
It is designed as a **class demonstration project** to highlight how AI tools can simplify and enhance HR activities.

---

Objective

* To demonstrate real-world applications of AI in HR
* To create professional HR content using AI tools
* To improve creativity, efficiency, and automation in HR tasks

---

 AI Tools Used

* Gemini

* Grok
